# Scientific Programming

This repository contains some C# code developed for the Scientific Programming class (PEL216) at [**Centro Universitário FEI**](https://portal.fei.edu.br/).

## Pre-requisites
To run this code, you will need:
* Visual Studio 2017 or greater.

## How to use
Is simple and easy to run this code. You just need to follow few steps that are listed bellow:

1. Clone/Download this repo to your PC.
2. If you do not have Visual Studio, download it from [Microsoft's website](https://visualstudio.microsoft.com).
3. Open the project's folder in Visual Studio. 
4. Run the Test project to execute the test cases.

You are ready to go now :+1:!

I hope you enjoy and learn as much as I did :punch:!
